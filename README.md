MyBatis-Spring适配器是MyBatis sql映射框架易于使用的Spring3桥接器。

支持版本
------------------

- 1.3.x - 继续支持Java 6
- master (2.0.x) - 支持 Java 8, Spring 5, 和 Junit 5 plus
